#FrameworkClinicaMedica
A importancia de tecnicas que aumentam a produtividade e a qualidade no ciclo de desenvolvimento de software
é um dos maiores obstaculos a serem superados pela engenharia de software. Desse modo, uma das abordagens para diminuir
a complexidade, aumentar a produtividade e a qualidade tem sido a proposição da reutilização do software.
